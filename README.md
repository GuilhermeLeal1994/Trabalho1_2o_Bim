# 📦 Sistema Controlador de Exercícios - TypeScript

Este repositório contém a resolução de 18 exercícios práticos de programação, 
cobrindo conceitos que vão desde a lógica de programação básica e manipulação de estruturas de dados até os pilares da Programação Orientada a Objetos (POO). 
A aplicação foi desenvolvida de forma robusta e otimizada, utilizando um painel de menu interativo via terminal com o uso de módulos nativos do Node.js.

## 🚀 Funcionalidades e Diferenciais do Projeto

* **Painel Categorizado:** Menu do terminal organizado inteligentemente por blocos de conhecimento (Lógica Básica, Arrays, Textos, POO, Utilitários).
* **Navegação Contínua:** Implementação de fluxo dinâmico permitindo que o usuário repita o cálculo atual ou retorne ao menu principal sem que o programa encerre.
* **Validação de Inputs:** Tratamento de erros integrado contra entradas de dados vazias ou do tipo texto onde se esperavam números.
* **Tipagem Estrita:** Uso pleno dos recursos de tipos e interfaces do TypeScript, garantindo a qualidade do código.
